# Journal Guide Template

Use this file to keep track of target venues, submission windows and formatting constraints. The `make journal-guide` target converts it to PDF, while `make journal-guide-docx` produces a DOCX with your reference styles.

## How to use

1. Duplicate the section below for every venue you care about.
2. Fill in the metadata (scope, impact, APC, timeline, contacts, links).
3. Capture submission tips, reviewer expectations and any custom formatting required.

### Why we care
- Aligns with our methodological focus on human-centered automation.
- Accepts articles in English or Italian with bilingual abstracts.
- Requires a structured abstract (Background, Methods, Results, Conclusions).

### Formatting checklist
- Max 8,000 words, 8 figures, 3 tables.
- Use APA references; upload BibTeX + RIS alongside the manuscript.
- Submit separate high-resolution figure files (PNG or PDF, 300 dpi min).
